## Carrossel vinculado ao scroll - Elementor + GSAP

### Estrutura:

- Container da seção com a classe **container-cards**
- Container interno com a classe **cards**
- Slides dentro do container interno com a classe **card**

Coloque o código css do `carrossel-vinculado-scroll.css` no CSS Personalizado;

| Coloque o código do `carrossel-vinculado-scroll.html` em um widget HTML, no rodapé da página.
